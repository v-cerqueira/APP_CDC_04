# Portal de Noticias Juridicas

Aplicativo mobile desenvolvido com React Native e Expo para apresentar um portal de noticias juridicas em uma interface simples, organizada e responsiva.

## Sobre o projeto

O app exibe:

- cabecalho com identidade visual do portal;
- secao "Leia Mais" com cards de noticias;
- rodape com acoes rapidas de navegacao.

Os dados exibidos atualmente sao estaticos e servem como base para estudos, prototipacao e evolucao futura da aplicacao.

## Tecnologias utilizadas

- React Native
- Expo
- JavaScript

## Como executar

1. Instale as dependencias:

```bash
npm install
```

2. Inicie o projeto:

```bash
npm start
```

3. Depois, escolha uma das opcoes no Expo:

- abrir no navegador;
- executar em emulador Android;
- executar em simulador iOS.

## Scripts disponiveis

- `npm start`
- `npm run android`
- `npm run ios`
- `npm run web`
